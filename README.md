# GramSahayak AI

GramSahayak AI is a multilingual AI-powered digital assistant designed 
to help rural communities access government schemes, healthcare information, 
agricultural support, and educational resources through simple voice and text 
conversations in Telugu, Hindi, and English.

This project is developed by **Team Avengers** for **AVISHKAAR - National Level Hackathon (Season 3)**.
